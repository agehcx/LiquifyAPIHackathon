# design/ — Index

Quick reference for all design system files. Read this first before opening any design file.

---

## Files

| File | What's Inside | When to Read |
|------|--------------|--------------|
| [`ci-brand.md`](./ci-brand.md) | Color tokens, typography scale, font rules, data visualization colors | Before writing any component with color or text |
| [`ui-theme.md`](./ui-theme.md) | Component patterns, shape language, spacing scale, surface system, motion timings, interaction states | Before writing any component |
| [`ux-principles.md`](./ux-principles.md) | Accessibility (WCAG AA), responsive breakpoints, performance targets, navigation UX, data/state display rules | Before building any page, layout, or interactive feature |
